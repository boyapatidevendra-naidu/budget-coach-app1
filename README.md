This application is a Smart Budget Coach built using Streamlit for interactive financial tracking.
It stores user transactions (income and expenses) in a SQLite database and retrieves them dynamically.
The dashboard calculates total balance, income, expenses, and savings in real time from stored data.
Visualizations like bar charts and pie charts are generated using Plotly to show spending patterns clearly.
Whenever a new transaction is added, the app refreshes automatically to update all metrics and charts.
Due to processing and rendering, there may be a delay of around 2 seconds before updated data appears in visuals.
The system ensures a smooth user experience with login-based sessions and continuously updated insights
